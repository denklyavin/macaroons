# Практическая работа: Верстка лендинга Macaroons

Этот репозиторий содержит результат выполненной практической работы по верстке.
Задача заключалась в создании лендинга для бренда "Macaroons" с использованием исключительно возможностей HTML и CSS, без привлечения сторонних библиотек.

## Что было сделано

- Сверстана статичная страница по макету.
- Реализованы интерактивные состояния кнопок и карточек (hover).
- Обеспечена адаптивность интерфейса до мобильных устройств.
- Проработана сетка:
  - **Grid Layout** использован для разбивки страницы на крупные блоки и вывода каталога товаров (меню).
  - **Flexbox** использован для управления положением элементов внутри блоков (шапка сайта, преимущества, расположение текста и изображения в секциях).

## Технологический стек

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)

## Как посмотреть

Просто открой файл `index.html` в браузере.
